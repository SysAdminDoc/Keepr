# Keepr Roadmap

## v0.1 (initial release)

- [x] Tauri 2 + React + TS + Tailwind scaffold
- [x] SQLite schema (notes, labels, note_labels, checklist_items)
- [x] CRUD for text + checklist notes
- [x] 12 Keep colors (light + dark)
- [x] Masonry grid with pinned/others sections
- [x] Note editor modal
- [x] Color picker, label chips, pin/archive/delete
- [x] Sidebar: Notes, Labels, Archive, Trash
- [x] Search (title, body, checklist text)
- [x] Manual ZIP export / import
- [x] Light / dark theme toggle
- [x] Portable Windows EXE build

## v0.2 (next)

- [ ] Image attachments (drag/drop, paste from clipboard)
- [ ] Reminders (in-app time-based, native Windows toast)
- [ ] Drag-and-drop reorder of notes
- [ ] Drag-and-drop reorder of checklist items
- [ ] Keyboard shortcuts (N for new, / for search, E for archive)
- [ ] Background image themes (Keep's 9 patterns)
- [ ] System tray quick-capture

## v0.3+

- [ ] Markdown rendering toggle
- [ ] Rich text formatting (bold, italic, links)
- [ ] Per-note password lock
- [ ] Auto-backup on schedule (daily ZIP to sync folder)
- [ ] Optional native Google Drive API sync (OAuth)
- [ ] Cross-platform builds (macOS, Linux)
